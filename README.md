# Pro Score Tracker

Pro Score Tracker is a modern and customizable web application designed to simplify score and point tracking. Whether you're monitoring exam performance, evaluating mock tests, tracking habits, managing game scores, or measuring productivity, this tool provides a fast and intuitive solution.

The application allows users to define custom positive and negative values, enabling it to adapt to different scoring systems. With a single click, scores are updated in real time, while built-in click counters provide additional insights into user activity.

## Features

- Real-time score calculation
- Customizable positive and negative values
- Click tracking and statistics
- Undo last action
- Reset functionality
- Automatic preference saving


## Purpose

The goal of this project is to provide a simple, efficient, and flexible score-tracking solution that anyone can use without technical expertise.
